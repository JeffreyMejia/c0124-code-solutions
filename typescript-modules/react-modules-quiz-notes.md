# react-intro-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are modules?
  A collection of code in an external file which performs a specific task or function.
- What is the difference between named and default imports?
  named imports must maintain the same name across all imports.
- What is the difference between named and default exports?
  for default exports the component can have any name. In named exports the name of the component must be maintained across all exports.
